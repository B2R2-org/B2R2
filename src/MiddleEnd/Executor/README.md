# B2R2.MiddleEnd.Executor

### B2R2?

B2R2 is a binary analysis and reversing framework written purely in F#. Since it
does not rely on any native (unmanaged) code, it is readily usable in any
platform or OS that .NET runs on.

### B2R2.MiddleEnd.Executor Package?

`B2R2.MiddleEnd.Executor` provides an execution engine that can execute the
intermediate representation (IR) generated by `B2R2.FrontEnd`.
