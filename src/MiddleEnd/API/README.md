# B2R2.MiddleEnd.API

### B2R2?

B2R2 is a fully managed binary analysis framework written in F#. It provides a
rich set of algorithms, functions, and tools for reverse engineering, program
analysis, and binary-level inspection.

### B2R2.MiddleEnd.API Package?

`B2R2.MiddleEnd.API` provides the main interface for the B2R2's
middle-end. In most cases, it should suffice to import only
`B2R2.MiddleEnd.API` to utilize our middle features.
